# ⚠️ Este repositório foi arquivado como fonte de desenvolvimento

## Status

Este repositório (`netops-ai-platform`) era o repositório público de desenvolvimento inicial.

A partir de **maio de 2026**, o desenvolvimento passou a ocorrer exclusivamente no repositório privado interno.

## Por que?

O modelo de dois repositórios simultâneos (público + interno) gerou fricção desnecessária:

- duplicidade de commits
- risco de divergência entre as duas linhas
- o código genérico e o de ambiente real precisam conviver durante o desenvolvimento

A separação será feita **ao final do projeto**, quando um processo de sanitização irá gerar
uma versão pública anonimizada a partir do repositório interno.

## O que está aqui

Este repositório contém a estrutura inicial e os commits de bootstrapping até a versão `0.3.0`.
Ele serve como referência histórica.

## Repositório ativo

O desenvolvimento ativo ocorre no repositório privado interno.
Quando a plataforma atingir maturidade suficiente, uma versão sanitizada será publicada aqui.
